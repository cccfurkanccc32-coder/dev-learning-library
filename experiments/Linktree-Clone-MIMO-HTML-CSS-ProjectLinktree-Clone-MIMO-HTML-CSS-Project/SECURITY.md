# Security Policy

## Supported Versions

This project is a simple static HTML/CSS learning project.  
There are currently no versioned releases. The latest version available in the repository is the only supported version.

## Reporting a Vulnerability

If you discover a security issue related to this project, please report it by opening an issue on this repository.

Please include:

- A clear description of the issue
- Steps to reproduce the vulnerability
- Any relevant screenshots or code references

Thank you for helping improve the security of this project.
